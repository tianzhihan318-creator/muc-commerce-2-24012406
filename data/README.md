# 数据文件说明

| 文件 | 用途 | 课程 |
|---|---|---|
| `淘宝全品类全国数据.csv` | 商品数据读取、筛选、排序和分组统计 | Day03 |
| `E Commerce Dataset.xlsx` | 电商用户数据清洗与预处理 | Day04～Day05 |

## 使用边界

- 数据仅用于课程实训；
- 不要修改或覆盖原始数据；
- Day04的清洗结果写入`output/day04_project/`；
- Day05只读取Day04生成的`ecommerce_customer_cleaned.csv`；
- 教师公开发布本仓库前应确认数据再分发许可，许可不明确时建议使用私有GitHub仓库。
